2. 데이터 분리와 카테고리화

group split + balanced subset

핵심 설명

  • 등급은 A/B/C로 통일했다.
  • 세부 각도 5개를 top/middle/side 3개 view로 재분류했다.
  • 같은 사과가 train/test에 동시에 들어가지 않도록 group_id 기준으로 나눴다.
  • Training + Validation을 합친 뒤 7:3, 다시 8:2로 분리했다.

3. 모델 구조와 학습 방식

ResNet18 전이학습

핵심 설명

  • PyTorch ResNet18에 ImageNet pretrained weight를 사용했다.
  • 마지막 fc layer만 class 수에 맞게 교체했다.
  • view 모델 1개가 top/middle/side를 먼저 판단한다.
  • 이후 view별 grade 모델 3개 중 하나가 A/B/C를 예측한다.

4. 성능 비교와 최종 모델 선정

핵심 설명

  • random split은 점수가 높지만 같은 사과 누수 가능성 때문에 제외했다.
